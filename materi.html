Apa itu Programming?

Programming adalah proses menulis, menguji, dan memelihara kode yang digunakan untuk memberi instruksi kepada komputer agar dapat menjalankan tugas tertentu sesuai dengan kebutuhan pengguna atau sistem. Proses ini melibatkan penggunaan berbagai bahasa pemrograman, algoritma, dan logika untuk menciptakan perangkat lunak yang fungsional dan efisien. Dengan programming, kita dapat mengembangkan berbagai jenis aplikasi, mulai dari perangkat lunak desktop, aplikasi mobile, hingga sistem berbasis web yang dapat digunakan untuk berbagai keperluan, seperti bisnis, pendidikan, hiburan, dan otomasi tugas-tugas kompleks. Selain itu, programming juga memungkinkan pengembangan sistem otomatisasi yang dapat meningkatkan efisiensi kerja, mengurangi kesalahan manusia, serta mempercepat penyelesaian berbagai proses dalam dunia industri dan teknologi modern.

Bahasa Pemrograman Web

Dalam semua ilmu tentang pemrograman baik itu desktop, mobile, game maupun software atau aplikasi lainnya pasti selalu ada bahasa-bahasa pemrograman yang digunakan untuk membuat aplikasi tersebut yang berisi statement-statement, perintah, atau pun fungsi agar aplikasi yang dibuat sesuai dengan apa yang telah didesain oleh si programmer. Oleh karena itu pemrograman web pun memiliki banyak bahasa yang digunakan, di antaranya:

Python

Digunakan untuk AI, Data Science, dan Web Development.

JavaScript

Digunakan untuk membuat website interaktif.

HTML

Digunakan untuk membuat struktur halaman web.

CSS

Digunakan untuk mendesain tampilan halaman web.

Python

Python adalah bahasa pemrograman tingkat tinggi yang pertama kali dikembangkan oleh Guido van Rossum pada tahun 1991. Bahasa ini dirancang dengan sintaks sederhana, mudah dipelajari, dan digunakan secara luas dalam pengembangan web, AI, machine learning, dan data science. Beberapa framework populer yang menggunakan Python adalah Django dan Flask.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website interaktif. Awalnya dikembangkan oleh Netscape pada tahun 1995, JavaScript kini menjadi bahasa yang penting dalam pengembangan frontend maupun backend menggunakan Node.js, dengan framework populer seperti React, Angular, dan Vue.js.

HTML

HTML sebenarnya bukan sebuah bahasa pemrograman, melainkan markup language atau bahasa penandaan yang terdiri dari kumpulan tag. Pada dasarnya HTML hanya mendeskripsikan bahwa bagian tertentu dalam sebuah halaman web adalah isi yang harus ditampilkan oleh browser dengan cara tertentu. HTML merupakan standar internet yang didefinisikan dan dikendalikan oleh World Wide Web Consortium (W3C).

CSS

CSS merupakan bahasa stylesheet yang digunakan untuk mengatur tampilan suatu dokumen yang ditulis dengan HTML. CSS juga memiliki css framework dan digunakan untuk menambah desain-desain tertentu pada halaman web agar desain halaman menarik untuk dilihat. Penggunaan CSS paling umum adalah untuk mengatur halaman web yang ditulis dengan HTML atau XHTML.
